
# Web Dashboard Inventaris Barang - Laravel

## Fitur
- Login Admin
- CRUD Barang
- CRUD Supplier
- Transaksi Barang Masuk
- Transaksi Barang Keluar
- Dashboard Grafik Stok

## Teknologi
- Laravel 10
- MySQL
- Bootstrap 5
- Chart.js

## Cara Menjalankan
1. Install Laravel
2. Copy file project ini
3. Import database `database/inventaris.sql`
4. Jalankan:
   ```bash
   php artisan migrate
   php artisan serve
   ```
